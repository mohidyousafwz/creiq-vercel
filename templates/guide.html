Welcome to CREIQ - Getting Started

This system helps you automatically collect appeal information from the ARB (Assessment Review Board) website without having to visit each property manually.

What does CREIQ do?

CREIQ saves you time by:

  • Automatically visiting the ARB website for each property roll number
  • Finding and collecting all appeal information for each property
  • Organizing everything in one easy-to-use dashboard
  • Allowing you to export all the data for your reports

How it works - Simple 3-step process:

1

Upload your list

Prepare a simple list of property roll numbers and upload it

2

Let CREIQ work

Watch as the system automatically visits the ARB website and collects all appeal data

3

Review and export

View the collected appeals and export everything for your analysis

Navigation Menu

Use the menu at the top to navigate between different sections:

  • Dashboard: Overview of all your data and system activity
  • Roll Numbers: Upload new properties and view extraction progress
  • Scraper Logs: See what the system is doing in real-time
  • Settings: Backup your data and manage the system
  • System Health: Check if everything is working properly
  • User Guide: This help page

Step 1: Uploading Your Roll Numbers

Before CREIQ can collect appeal data, you need to tell it which properties to look up by uploading a list of roll numbers.

Preparing Your CSV File

A CSV file is like a simple text file that contains your list of roll numbers. Here's how to create one:

Option A: Using Excel or Google Sheets
  1. Open Excel or Google Sheets
  2. Put each roll number in column A, one per row (no headers needed)
  3. Save the file as "CSV" format
  4. Choose a simple name like "my-roll-numbers.csv"
Option B: Using Notepad
  1. Open Notepad (Windows) or TextEdit (Mac)
  2. Type each roll number on its own line
  3. Save the file with ".csv" at the end of the name
Your file should look like this:
38-29-300-012-10400-0000
06-14-041-701-16500-0000
19-08-072-215-00500-0000
‚ö†Ô∏è Important: Roll Number Format

Make sure your roll numbers follow this format: XX-XX-XXX-XXX-XXXXX-XXXX

Example: 38-29-300-012-10400-0000 (includes all the dashes)

Upload Process - Step by Step

  1. Go to Roll Numbers page: Click "Roll Numbers" in the top menu
  2. Start upload: Click the blue "Upload CSV" button at the top of the page
  3. Choose your file: Click "Choose File" and select your CSV file from your computer
  4. Upload the file: Click the "Upload" button
  5. Review the summary: The system will tell you:
    • How many roll numbers were found
    • How many are new vs. already in the system
    • Any errors if roll numbers are in wrong format
  6. Choose what to do next:
    • "Process All" - Start collecting appeal data automatically
    • "I'll do Manually" - Just add to your list, process later
‚úÖ What happens after upload

Your roll numbers are immediately added to your list and will appear in the table below. If you chose "Process All", CREIQ will start visiting the ARB website to collect appeal information automatically.

Step 2: Watching CREIQ Work - Monitoring Progress

Once you click "Process All", CREIQ starts working automatically. Here's what to expect and how to monitor the progress.

What Happens During Extraction

For each roll number in your list, CREIQ will:

  1. Open a web browser (you won't see this - it runs in the background)
  2. Go to the ARB website (arbesc.arb.gov.on.ca)
  3. Enter the roll number into the search form
  4. Search for appeals related to that property
  5. Collect all appeal details including dates, decisions, and documents
  6. Save everything to your database for later review

What it means: CREIQ is currently working on this property

What it means: All appeals for this property have been collected successfully

Failed

What it means: Something went wrong (see troubleshooting section below)

Check the Scraper Logs for details about what happened

How Long Does It Take?

Processing time depends on how many appeals each property has:

  • ‚Ä¢ No appeals: ~10 seconds per property
  • ‚Ä¢ 1-3 appeals: ~30-60 seconds per property
  • ‚Ä¢ 5+ appeals: ~2-3 minutes per property

Example: 10 properties with an average of 2 appeals each = about 7-10 minutes total

Troubleshooting & Common Questions

Having issues? Here are solutions to the most common problems users encounter.

Upload Problems

‚ùå "No valid roll numbers found" error

What this means: CREIQ couldn't read any properly formatted roll numbers from your file.

  • Check that each roll number follows this format: XX-XX-XXX-XXX-XXXXX-XXXX
  • Make sure you include all the dashes (there should be 5 dashes)
  • Put only one roll number per line
  • Don't include headers like "Roll Number" at the top
  • Save your file as CSV format, not Excel (.xlsx)

Extraction Problems

‚ùå Roll number shows "Failed" status

Step 1: Check the Scraper Logs

  1. Go to "Scraper Logs" in the main menu
  2. Look for error messages related to your roll number

Common errors:

  • "Failed to submit search": ARB website might be down temporarily
  • "Invalid roll number": The roll number format is wrong
  • "Network error": Internet connection problem
  • "No property found": Roll number doesn't exist in ARB system

Step 2: Try processing again by clicking the "Process" link next to the failed roll number.

‚è≥ Processing seems stuck or very slow

This is usually normal! Processing can be slow because:

  • Properties with many appeals take longer (each appeal processed individually)
  • ARB website responds slowly during busy periods
  • Your internet connection might be slow

What to do: Be patient and check progress messages. If stuck for more than 10 minutes, check Scraper Logs for errors.

General System Issues

üîÑ Page won't load or shows errors

Quick fixes:

  1. Refresh the page (press F5 or click refresh button)
  2. Clear your browser cache (Ctrl+Shift+R)
  3. Try a different browser (Chrome, Firefox, Edge)
  4. Check if you're still logged in

Getting More Help

üí° Best Practices to Avoid Problems
  • Always test with a small file (2-3 roll numbers) before processing large batches
  • Keep your browser window open during processing
  • Regularly backup your data using the Settings page
  • Don't run multiple extractions at the same time
  • Check the System Health page before starting large processing jobs
